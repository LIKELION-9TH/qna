#### 📌 요약
- Live Server 켜는 단축키: **html 파일을 선택**한 후,
Window: `Alt + L + O`
Mac: `Option + L + O`

체크해야할 사항
[] **html파일**로 **저장**을 했는지! 
ex) test.html(이처럼 확장자가 html 파일이어야 한다)
